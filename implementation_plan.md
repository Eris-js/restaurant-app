# Restaurant Website Implementation Plan

## Goal
Build a complete, responsive restaurant website.

## Tech Stack
- **Framework**: Next.js 14+ (App Router)
- **Styling**: Tailwind CSS
- **Database**: MongoDB (via Mongoose)
- **Menu Effect**: 3D Flipbook (react-pageflip)

## Status Update
- **Navbar/Footer**: Completed
- **Landing Page**: Ported (Components: Hero, About, Reviews, Map)
- **Database**: Schemas ready, Connection ready.
- **Gallery**: Implementing "Instagram-style" Grid with Tabs & Lightbox.
- **Menu**: Implementing 3D Flipbook using images from `public/menu`.

## Upcoming Tasks
1.  **Backend APIs**: CRUD for content.
2.  **Admin Dashboard**: To upload images/manage sliders.
3.  **Authentication**: For Admin access.
